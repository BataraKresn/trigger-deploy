üîê Authentication

All protected endpoints require a valid token. The token can be provided via:

  • Query Parameter: ?token=YOUR_TOKEN
  • POST Body: {"token": "YOUR_TOKEN"}
  • Authorization Header: Authorization: Bearer YOUR_TOKEN
# Current token (from .env) TOKEN = SATindonesia2026

üöÄ Deployment Endpoints

GET POST /trigger

Trigger Deployment

Triggers a deployment process on the server or specified target.

Parameters:
token (required) - Authentication token
server (optional) - Target server IP address
Example Request:
GET /trigger?token=SATindonesia2026 POST /trigger { "token": "SATindonesia2026", "server": "192.168.1.100" }
Response:
{ "status": "success", "deployment_id": "deploy-20250720-123456", "log_file": "trigger-20250720-123456.log", "view_log_url": "https://yourdomain.com/logs/trigger-20250720-123456.log", "stream_log_url": "https://yourdomain.com/stream-log?file=trigger-20250720-123456.log", "timestamp": "2025-07-20T12:34:56", "server_name": "Production Server" }
POST /ping

Ping Server

Test SSH connectivity to a configured server.

Request Body:
{ "server": "192.168.1.100" }
Response:
{ "status": "ok", "server": "192.168.1.100", "timestamp": "2025-07-20T12:34:56" }

üîç Service Monitoring

GET /api/services/status

Get Services Status

Get current status of all monitored services including Docker containers and remote services.

Response:
{ "services": { "local": [...], "remote": [...] }, "summary": { "total": 5, "healthy": 4, "unhealthy": 1, "critical_down": 0 }, "monitoring_config": { "interval": 60, "timeout": 30, "enabled": true, "alert_cooldown": 300 } }
GET /api/services/config

Get Services Configuration

Retrieve the current services monitoring configuration.

POST /api/services/config

Update Services Configuration

Update the services monitoring configuration. Requires authentication.

Headers:
Authorization: Bearer SATindonesia2026

üìä Metrics & Analytics

GET /metrics

Basic System Metrics

Get basic system health and deployment metrics.

Response:
{ "status": "healthy", "uptime_seconds": 86400, "active_deployments": 2, "total_logs": 45, "memory_usage_percent": 65.2, "disk_usage_percent": 23.1, "timestamp": "2025-07-20T12:34:56" }
GET /api/metrics/stats

Deployment Statistics

Get detailed deployment statistics and history.

GET /api/metrics/history

Deployment History

Get recent deployment history with optional limit.

Parameters:
limit - Number of records (default: 20)

üìã Log Management

GET /logs

List Available Logs

Get list of all available log files.

GET /log-content

Get Log Content

Get content of a specific log file.

Parameters:
file - Log filename
Example:
GET /log-content?file=trigger-20250720-123456.log
GET /stream-log

Stream Log (Real-time)

Stream log content in real-time using Server-Sent Events.

Parameters:
file - Log filename
